# The Virtual Guy

The 2D Platformer Advanture game.


![Image description](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/a4ej1924k8azvthg4i2e.png)


## Features

- Movement, Jump 
- Character Animation
- Fullscreen mode
- Traps and collective items
- WebGL Support
